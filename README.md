# Stellar Server with NodeJS

## Running Locally

Make sure you have [Node.js](http://nodejs.org/) installed.

```sh
$ git clone https://github.com/azzuarezh/stellar-server-node.git # or clone your own fork
$ cd stellar-server-node
$ npm install
$ node index.js
```
